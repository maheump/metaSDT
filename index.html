metaSDT

Individual signal detection analysis for type II data.

Description.

metaSDT is a R function analyzing your type II data with signal detection theory. It works with confidence ratings (any type of scale with 1 step between two grades) and control levels (such as information seeking level or amount of exploration).

Input.

It only requires:

  • an "input_data" data.frame with answers labels in the first column and confidence ratings in the second one,
  • and confidence borns.

It can also take into account design (detection or choice), ponderation type, and homogeneity analysis, which are specified by default.

Answers labels.

Answers have to be classified (answers labels) following classical SDT categories:

  • 1: Hit,
  • 2: False alarm,
  • 3: Miss,
  • 4: Correct rejection.

Utilization.

  • Open R.
  • Set the working directory to the right place.
setwd("~/Documents/R/metaSDT")
  • Load the function.
source("MM_metaSDT.r")
  • Load the example data.
data <- read.csv("example_data.csv", header = TRUE, sep = ";")
  • Launch the function.
MM_metaSDT(data, 1, 6)
